Salesforce

We support synchronizing data between Salesforce and Crowdstart. This document tries to outline how our integration works and what you can expect when using it.

Currently Crowdstart supports synchronizing the following Salesforce objects to/from our platform:

  • Account
  • Contact
  • Order
  • Order Products
  • Price Book
  • Product

Salesforce to Crowdstart mapping

Accounts

Salesforce Accounts map to Crowdstart User objects. It contains shipping and billing information for the User.

Field label Field name Crowdstart mapping Type
Billing Address BillingAddress User.BillingAddress Address
Shipping Address ShippingAddress User.ShippingAddress Address

Contacts

Salesforce Contacts map to Crowdstart User Objects. It contains Contact information for the User. Each Contact is linked to an Account

Field label Field name Crowdstart mapping Type
Last Name LastName User.LastName Text(80)
First Name FirstName User.FirstName Text(40)
Email Email User.Email Email
Phone Phone User.Phone Phone
Account Name Account User.Key Lookup(Account)

Products

Salesforce Products map to Crowdstart product variants.

Field label Field name Crowdstart mapping Type
Product Code ProductCode Variant.SKU Text(255)
Product Description Description Product.Description Text Area(4000)
Product Name Name Product.Name Text(255)

Price Books

A Salesforce Price Book is a catalog of products. Our integration adds products automatically to the standard Salesforce Price Book.

Orders

A Salesforce order is mapped out to a Crowdstart order fairly closely. We've added several extra fields to make it easier to understand the status of your order.

Standard fields

Fields that we update that are part of the standard Salesforce Order object.

Field label Field name Crowdstart mapping Type
Account Name Account User.Key Lookup(Account)
Billing Address BillingAddress Order.BillingAddress Address
Description Description Order.Description TextArea(32000)
Order Amount TotalAmount Order.Subtotal Currency(16,2)
Order Name Name Order.Key Text(80)
Order Start Date EffectiveDate Order.CreatedAt Date
Shipping Address ShippingAddress Order.ShippingAddress Address

Note: Order Amount is generated by Salesforce. It's the sum of Order Product's unit price times quantity. As a result, it will not always match the true subtotal and always is missing shipping/tax information. You should prefer our custom fields in almost all cases.

Order Products

A salesforce order product roughly maps to a single line item in a Crowdstart order.

Standard fields

Fields that we update that are part of the standard Salesforce Order Product object.

Field label Field name Crowdstart mapping Type
Line Description Description LineItem.Description Text(255)
Order Order Order.Key Lookup(Order)
Product Product2 Product.Key Lookup(Product)
Product Code ProductCode LineItem.SKU Text(255)
Quantity Quantity LineItem.Quantity Number(16, 2)
Start Date ServiceDate Order.CreatedAt Date
Unit Price UnitPrice LineItem.Variant.Price Currency(16, 2)

Custom fields

Fields that we've added to Salesforce Order Product object to make synchronization simpler.
