Getting Started

Welcome to the Crowdstart documentation. Looking to get up and going fast? Checkout Crowdstart.js. Building a custom integration? Get up and running with our API documentation.

You can view documentation here, or on our public wiki.

Support

Having problems with the platform? Hit us up at support@crowdstart.com or submit a new issue.

Open Source

We've open-sourced several parts of the Crowdstart platform. We've made these parts of Crowdstart available to the public under an MIT license.

Crowdstart.js

Our easy to use client-side library.

Checkout

Checkout is an easy to use JavaScript widget that makes it possible for you to start taking pre-orders in minutes.

Crowdstart CLI

A command-line interface to Crowdstart.

Hustle

A JavaScript animation library.

Quill

A functional CSS library.
