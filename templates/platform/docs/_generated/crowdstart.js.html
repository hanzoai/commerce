Crowdstart.js

Crowdstart.js makes it possible to authorize and charge orders directly on your site. This makes it possible to write more sophisticated integrations and completely customize the look and feel of your site.

We've made the code open source, so fork away. If you build anything cool with it, let us know :)

If you need any help or discover any issues, you can report them on our issue tracker.

Payment flows

Crowdstart supports two different payment flows. For most people, you'll want to use our simple Charge payment flow. This lets you authorize and immediately capture payment.

For more complicated use-cases, you'll want to use a full two-step payment flow, commonly referred to as as "Auth and Capture".

Integrating Crowdstart.js

<script type="text/javascript" src="https://js.crowdstart.com/v1.js"></script>

Add this script tag to your site to get started.

Setting your publishable key

You must set your publishable key with Crowdstart.setKey to identify your site to Crowdstart. You can find your publishable key in your settings.

Crowdstart.setKey('your-publishable-key');

Collecting payments

You can use Crowdstart.charge to authorize and immediately capture payment from your users.


Crowdstart.charge({
  "payment": {
    "type": "stripe",
    "account": {
      "number": "4242424242424242",
      "month": "12",
      "year": "2016",
      "cvc": "123"
    },
    "metadata": {
      "paid": "in full"
    }
  },
  "user": {
    "email": "suchfan@shirtlessinseattle.com",
    "firstName": "Sam",
    "LastName": "Ryan",
    "company": "Peabody Conservatory of Music",
    "phone": "555-555-5555",
    "address": {
      "line1": "12345 Faux Road",
      "city": "Seattle",
      "state": "Washington",
      "country": "United States",
      "postalCode": "55555-5555"
    },
    "metadata": {
      "sleepless": true
    }
  },
  "order": {
    "currency": "usd",
    "items": [{
      "productSlug": "doge-shirt",
      "price": 100,
      "quantity": 2
    }],
    "metadata": {
      "shippingNotes": "Ship Ship to da moon."
    }
  }
}, function(status, response) {})

This will create a new order, optionally associated with a customer on our end, authorize and capture payment.

Two-step payment flow

Rather than authorize and capture all in one go, you can use Crowdstart.authorize to save an authorization, and charge later when you ship or when your campaign completes, etc.
